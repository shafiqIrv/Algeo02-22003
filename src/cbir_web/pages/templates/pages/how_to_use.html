Guide to use this tool

  1. Klik tombol “Upload Dataset” dan pilih dataset gambar dalam bentuk folder yang berisi kumpulan gambar.
  2. Drag file gambar yang ingin kamu cari ke “Insert Image Here”.
  3. Pilih opsi pencarian berdasarkan warna atau tekstur.
  4. Tekan tombol search untuk mencari gambar.
  5. Gambar telah selesai dicari.
